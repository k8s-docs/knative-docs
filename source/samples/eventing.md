# Knative 事件代码示例

使用下面的代码示例来帮助您理解 Knative 事件和事件源的各种用例。

[了解更多关于 Knative 事件和事件来源](../eventing/README.md).

参见 GitHub 中的[所有 Knative 代码示例](https://github.com/knative/docs/tree/main/code-samples)。
