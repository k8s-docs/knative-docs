`build` 命令使用项目名称和镜像注册表名称为函数构造一个完全限定的容器镜像名称。
如果之前没有构建该函数项目，则会提示您提供一个镜像注册表。

=== "func"

    要构建该函数，运行以下命令:

    ```bash
    func build
    ```

=== "kn func"

    要构建该函数，运行以下命令:

    ```bash
    kn func build
    ```
