    更新 Knative¶

    Knative 支持通过单个minor版本号进行升级。 例如，如果您安装了 v0.21.0，那么在尝试升级到 v0.23.0 之前，您必须升级到 v0.22.0。 要验证当前版本，请参见检查 Knative 版本。

    升级 Knative:

    • 如果您使用 YAML 安装 Knative，请参见用 kubectl 升级.
    • 如果您使用 Knative 操作符安装 Knative，请参阅使用 Knative Operator 升级.
